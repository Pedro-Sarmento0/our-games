# nossos-jogos

A collection of simple games built with HTML, CSS, and JavaScript. This project was created to practice programming logic, and game development concepts.

## Games

* 🐍 Snake
* 🃏 Memory Game
* 🔤 Hangman
* 🎵 Simon Says

## Technologies

* HTML
* CSS
* JavaScript

## What I Practiced

* Event handling
* Arrays and objects
* Functions
* Game logic
* Keyboard and mouse interactions

* Make the games responsive for mobile devices.
* Add more games.
